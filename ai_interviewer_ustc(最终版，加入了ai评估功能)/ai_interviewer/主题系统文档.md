## 主题介绍

### 🌌 主题 1：赛博训练舱

**设计理念**：像进入一个未来科技的面试模拟舱，充满仪式感和专注力。

#### 视觉特征
- **配色**：深蓝黑 (#0a0e27) + 霓虹青 (#00f5ff) + 紫粉 (#b24bf3)
- **背景**：流动的青紫色极光 + 淡淡的网格线
- **质感**：磨砂玻璃卡片 + 内发光边框
- **装饰**：FUI 风格的角落装饰线条
- **字体**：Orbitron（标题）+ Inter（正文）+ Noto Sans SC（中文）

#### 核心动效
```
✨ 背景极光缓慢流动（20秒循环）
✨ 对话卡片从左右滑入 + 淡入
✨ 按钮悬停时波纹扩散效果
✨ FUI 装饰线条
✨ 标题霓虹发光
```

#### 适合场景
- ✅ 长时间面试练习（视觉舒适）
- ✅ 深夜独自训练（深色护眼）
- ✅ 需要专注但不失科技感
- ✅ 展示给朋友看（平衡美观与专业）

**情绪定位**：沉浸、专业、未来科技

### 🔧 核心功能

#### 1. 主题管理系统
```python
# themes/__init__.py
from . import theme_cyber, theme_neon

AVAILABLE_THEMES = {
    "赛博训练舱": theme_cyber,
    "霓虹实验室": theme_neon,
}

def get_theme_css(theme_name):
    """获取指定主题的 CSS"""
    theme = AVAILABLE_THEMES.get(theme_name)
    if theme:
        return theme.CSS_STYLE
    return AVAILABLE_THEMES["赛博训练舱"].CSS_STYLE
```

#### 2. 主题切换逻辑
```python
# app_streamlit.py
# 初始化主题
if "selected_theme" not in st.session_state:
    st.session_state.selected_theme = "赛博训练舱"

# 应用主题 CSS
st.markdown(get_theme_css(st.session_state.selected_theme), unsafe_allow_html=True)

# 主题选择器
theme_choice = st.selectbox(
    "选择界面风格",
    options=get_theme_list(),
    index=get_theme_list().index(st.session_state.selected_theme),
)

# 切换主题时重新加载
if theme_choice != st.session_state.selected_theme:
    st.session_state.selected_theme = theme_choice
    st.rerun()
```

#### 3. CSS 注入方式
- 所有主题通过 `st.markdown()` 注入自定义 CSS
- 无需外部 CSS 文件
- 支持动态切换

#### 4. 动画实现
- 使用 CSS3 `@keyframes` 动画
- GPU 硬件加速（`transform` 和 `opacity`）
- 避免重绘和回流
- 流畅度 60fps

### ⚡ 性能优化

- ✅ 使用 `transform` 和 `opacity`（GPU 加速）
- ✅ 避免使用 `width`、`height`、`top`、`left` 动画
- ✅ 背景动画使用 `position: fixed` 减少重绘
- ✅ 合理的 z-index 层级管理
- ✅ 响应式设计，适配不同屏幕

### 🌐 兼容性

- ✅ Chrome/Edge（推荐）
- ✅ Firefox
- ✅ Safari
- ⚠️ IE（不支持）

## 自定义指南

### 🎨 调整配色

编辑 `themes/theme_*.py` 文件，搜索颜色代码并替换：

```css
/* 例如：赛博训练舱的霓虹青 */
color: #00f5ff;  /* 改成你喜欢的颜色 */
```

### ⏱️ 调整动画速度

找到 `animation` 属性，修改时间参数：

```css
/* 例如：极光流动速度 */
animation: aurora-flow 20s ease-in-out infinite;
                      ↑
                    调整这个数字（秒）
```

### 🚫 关闭某个动画

注释掉对应的 `animation` 行：

```css
/* animation: aurora-flow 20s ease-in-out infinite; */
```

### ➕ 添加新主题

1. 复制现有的 `theme_*.py` 文件
2. 修改配色和样式
3. 在 `themes/__init__.py` 中注册：

```python
from . import theme_cyber, theme_neon, theme_new

AVAILABLE_THEMES = {
    "赛博训练舱": theme_cyber,
    "霓虹实验室": theme_neon,
    "新主题": theme_new,  # 添加这行
}
```

### 📏 调整字体大小

修改 CSS 中的 `font-size` 属性：

```css
p, .stMarkdown {
    font-size: 16px;  /* 调整这个值 */
}
```

## 常见问题

### Q: 切换主题后页面会刷新吗？
**A**: 是的，这是正常的。Streamlit 需要重新加载 CSS。

### Q: 主题会保存吗？
**A**: 会的！你选择的主题会保存在 session 中，刷新页面后仍然生效。

### Q: 字体加载慢怎么办？
**A**: 字体通过 Google Fonts CDN 加载，需要网络连接。如果网络不好，会自动回退到系统默认字体。

### Q: 动画在低端设备上卡顿怎么办？
**A**: 可以选择赛博训练舱主题（动画较少），或者按照自定义指南关闭部分动画。

### Q: 主题在手机上显示正常吗？
**A**: 所有主题都是响应式设计，在手机、平板、电脑上都能正常显示。
